# FilePicker
File and folder picker dialog

## Gradle:

  `compile 'com.github.isabsent:filepicker:1.0.00'`

## Maven:

    <dependency>
      <groupId>com.github.isabsent</groupId>
      <artifactId>filepicker</artifactId>
      <version>1.0.00</version>
      <type>pom</type>
    </dependency>
